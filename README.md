PulseCheck is a modern Next.js application for managing employee feedback, engagement, and analytics. It features AI-powered report generation, real-time notifications, activity logging, export functionality, and advanced search/filter tools. Built with React, Prisma, Tailwind CSS, and Genkit AI.

## üöÄ Features

- **AI Report Generation**: Generate personalized employee reports using Genkit AI.

- **Search & Filter**: Powerful, real-time search and multi-criteria filtering.
